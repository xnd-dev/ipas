# :inbox_tray: Installation 

on terminal, execute the command:

```bash
npm install
```

# :runner: Running project
on terminal, execute the command:
```bash
npm run dev
```

# :file_folder: Structure Folders
the structure folders of this project is defined like example above: 

```
src/
├── pages/
│   ├── products.tsx

├── features/

│   │   ├── layouts
```
